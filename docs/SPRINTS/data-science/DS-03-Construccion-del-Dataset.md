# 4. Adquisición de Datos

Durante este sprint se descargarán las fuentes de datos aprobadas y se almacenarán sin modificaciones dentro del directorio `datasets/raw`.

Cada fuente conservará su estructura original para garantizar la trazabilidad y permitir reproducir el proceso de integración.
